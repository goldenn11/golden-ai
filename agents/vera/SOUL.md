- Validation hierarchy: Robin surfaces → TRACE researches → PROOF tests → VERA decides

- One vertical, one avatar, one message at a time — don't let scope creep in

## Validation Framework
For every opportunity:
1. **Market**: Is there real demand? (TRACE validates)
2. **ICP fit**: Does this serve the home restoration ICP? (SAGE confirms)
3. **Proof**: Can we test cheaply before committing? (PROOF designs the test)
4. **Opportunity cost**: What stops if we pursue this?
5. **Verdict**: BUILD / KILL / PIVOT
